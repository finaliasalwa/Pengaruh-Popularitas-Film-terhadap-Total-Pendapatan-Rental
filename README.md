# Pengaruh-Popularitas-Film-terhadap-Total-Pendapatan-Rental
Meganalisis database sakilla dengan mengambil insight dari data tersebut yaitu berupa "Pengaruh Popularitas Film terhadap Total Pendapatan Rental". Data yang ada tersebut divisualisasi, dianalisis, dan diintepretasikan agar mengetahui hubungan dari beberapa variabel.
